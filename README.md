# THE DRAFT · speculative decoding

> A sphere of **UD0** — the ROOT0 universe / biosphere. Domain: **A SCANNER DARKLY**.

THE DRAFT — a cheap draft guesses a whole run of tokens; the big model checks them all at once and keeps the longest prefix it agrees with. A live instrument of SPECULATIVE DECODING (Leviathan et al.; Chen et al., 2023): accept a drafted token with probability min(1, p/q); on the first rejection resample from the residual (p−q)₊ and stop; if the whole draft survives, a bonus token comes free. The punchline, proven live: the emitted tokens are drawn from the target's EXACT distribution at any speculation length and any draft quality — free speed, provably no drift.
